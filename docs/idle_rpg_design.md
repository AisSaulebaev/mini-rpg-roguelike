# Idle RPG «Восхождение» — дизайн-документ (v0.3)

**Референс:** Path of Kings (TornadoBear) — endless gear-RPG, полностью idle-бой.
**Платформа:** Telegram WebApp. Прогресс **сервер-авторитетный** (на потом; прототип на localStorage).
**Жанр/вид:** киберпанк idle-RPG, **2.5D перспектива** (2D-спрайты + псевдо-3D земля, как в референсе), AI-спрайты (пайплайн из BD).

> v0.2 — убрано перерождение; снаряжение не затачивается; кредиты в уровень; Ядра → Магазин.
> v0.3 — после анализа скринов референса: **команда из 3 героев** (все дамагеры, разные профили) вместо 1 класса; **8 слотов на каждого героя**; «уровень» предмета = глава дропа; дроп отдаёшь одному из 3; взяты дроп-таблица % per-локация, сундуки-вехи, экран-карта кампании.
> v0.4 — после мокапа от веб-дизайнера (`Восхождение.html`): бой = **2.5D команда из 3** (переделан из FP). Пользователь ВЕРНУЛ: **престиж** (заверши карту N), **боссы ПОРЯДОК/ХАОС** (morality-шкала + призыв босса), **P2W-магазин** (VIP, без рекламы, наборы). Навигация = **5 вкладок**: Герои · Карта · Бой(центр) · Боссы · Магазин. Данжи — пока не в навигации (TBD). Две арт-темы ICE/PULSE.

## 0. Сеттинг и нарратив (киберпанк)

Дерёмся с **роботами/машинами**, не с людьми. Город будущего, связь захватила система машин-глушилок, игрок пробивается вверх.

**ЗОЛОТОЕ ПРАВИЛО БЕЗОПАСНОСТИ (не нарушать):**
- Никаких реальных ведомств/аббревиатур/стран/законов/политиков/флагов/геральдики.
- Враги — **безликие машины** (дроны, турели, боты), НЕ люди в форме, НЕ карикатуры.
- В текстах — никаких «обход блокировок» и инструкций. Игра — метафора.
- Тест: убери подтекст — читается как обычный киберпанк «человек против злой системы машин».

**Локация = блок из 3 глав**, своя визуалка + пул врагов:

| Локация | Главы | Враги | Босс локации |
|---|---|---|---|
| Трущобы / улица | 1–3 | дрон-глушилка, шокер-бот | Дрон-надзиратель |
| Станция связи | 4–6 | ремонт-бот, антенна-турель | 🤖 Робот-роутер |
| Дата-центр | 7–9 | серверный паук, охранный бот | Сервер-страж |
| Метро / туннели | 10–12 | турель, кабель-змея | Файрвол-состав |
| Небоскрёб | 13–15 | элитная охрана-меха | Мейнфрейм |
| ...рескины + рост чисел | 16+ | | |

## 1. Ощущение и петля

**Формула азарта:** дешёвый быстрый рост вначале → цена уровня героев растёт по экспоненте → упёрся в стену → **открываешь нового героя** (качаешь с нуля — ранний дофамин снова) + гонишься за шмотом/сундуками. «Свежий старт» дают **новые герои**, а не сброс прогресса.

### Core loop

```
Команда из 3 героев авто-дерётся с врагами главы (10 врагов; 10-й = элита; конец блока = босс)
     ↓  убийство
Шанс на предмет 🃏  (+ прочее падает)
     ↓
ОТДАЛ предмет одному из 3 героев / ПРОДАЛ лишнее → Кредиты 💰₡
     ↓
Прокачал УРОВНИ ГЕРОЕВ за кредиты (у каждого свой профиль статов)
     ↓  команда сильнее → добил босса → новая локация
Уперся в стену
     ↓
Данжи (Ядра + редкий шмот) · Магазин (сундуки/герои за Ядра) · Новый герой (свежий грайнд)
```

**Смерть (HP=0):** откат **на 1 главу назад** (не game over). Докачался — пошёл снова.

## 2. Статы (за кнопкой [!], не качаются напрямую)

Статы = производные от **уровня героя** + **его надетого снаряжения** (у каждого из 3 — свои). Отдельных кнопок-апгрейдов НЕТ. На боевом экране — кнопка `[!]`, тап → модалка со сводкой (по каждому герою).

| Стат | Что делает |
|---|---|
| DMG · AS · HP · ARM · REG | урон, скор.атаки, здоровье, броня, реген |
| Крит-шанс · Крит-урон | шанс ×множитель крита |
| Пробитие · Вампиризм · Уклонение · Шипы | продвинутые (в осн. со шмота) |
| LUCK (удача) · GREED (жадность) | качество дропа · множитель кредитов при продаже |

**Производные:** `DPS = DMG × AS × крит`; `dmgTaken = enemyDMG × 100/(100+ARM)`.

## 3. Главы и враги (формулы роста)

`c` — глава (с 1), `k` — враг в главе (1..10). Локация = `ceil(c/3)`.

```
enemyHP(c,k)  = 10 × 1.35^(c-1) × (1 + 0.05·(k-1))
enemyDMG(c,k) = 2  × 1.28^(c-1) × (1 + 0.04·(k-1))
sellValue(c)  = 5  × 1.22^(c-1) × GREED        // кредиты за продажу дропа этой главы
elite(c):  HP ×3, DMG ×1.5                      // 10-й враг обычной главы
boss(c):   HP ×8, DMG ×2.5, +Ядра              // 10-й враг главы 3/6/9... → новая локация
```
> Числа — черновые, тюним на прототипе Фазы 0.

---

## 4. Снаряжение 🃏 (главная ось силы, у КАЖДОГО героя своё)

Падает с врагов (шанс ~8%, растёт от LUCK) + из сундуков Магазина. **Не затачивается** — сила предмета фиксирована роллом. Дроп → **выбираешь, кому из 3 героев отдать** (или продать за кредиты). У каждого героя свой набор.

### 4.1 Слоты — 8 на героя (как в референсе)

`🔫 Оружие · 🧥 Броня · ⛑️ Шлем · 🧤 Перчатки · 👢 Ботинки · 🧿 Амулет · 💍 Кольцо · 🐾 Питомец`

3 героя × 8 = 24 слота всего. (Киберпанк-флавор: имплант вместо кольца, дрон вместо питомца — но слотов 8.)

### 4.2 «Уровень» предмета = тир силы = глава дропа

Как в референсе (там «Level 141» = предмет с ~14-й секции). Мы **не качаем предмет вручную** — его сила зашита в дроп:
```
itemLevel = глава, где выпал        // маркировка силы
rollPower ∝ 1.12^itemLevel          // статы предмета
```
Упал глубже → сильнее. Показываем «ур. N» на карточке предмета.

### 4.3 Редкости и аффиксы

`common · rare · epic · legendary · mythic` — множитель `×1 / ×1.6 / ×2.6 / ×4.2 / ×7`. Кол-во аффиксов растёт с редкостью (1 → 4–5).

**Таблица дропа per-локация (как в референсе):** каждая локация — свои % редкостей, и **чем дальше — тем лучше шансы** (ранняя: Необычный 70/Редкий 25/Эпич 5; дальше: 50/45/10; и т.д.).

Аффиксы: базовые статы + продвинутые боевые + **статус-эффекты**:
| Эффект | Тема | Механика |
|---|---|---|
| ⚡ Разряд | электрошок | оглушение 0.5–1с + прыжок на соседа |
| 🩸 Износ | коррозия | DoT, стакается |
| 🔥 Перегрев | поджиг | сильный DoT, обновляется |
| ❄️ Заморозка | крио | замедляет AS; добивание = +урон |
| 💥 ЭМИ | EM-импульс | отключает робота |
| 🦠 Вирус | заражение | +% входящего урона, заражает соседей |

Формат аффикса статуса: `шанс% + сила + длительность`. Создают билды (DoT / контроль / бёрст).

### 4.4 Взаимодействие
Дроп → карточка предмета → **сравнение** с надетым у каждого героя (▲зелёным/▼красным) → **отдать герою 1/2/3** или **продать** (→ кредиты). Авто-продажа мусора ниже выбранной редкости.

## 5. Герои 🦸 (команда из 3, грайнд за кредиты)

**Команда из 3 героев** сражается вместе (слева в side-view). Есть **ростер** — открываешь героев и ставишь троих в команду. Каждого **качаешь за кредиты, поуровнево** — основной грайнд и стена.

```
lvlCost(hero,L) = 50 × 1.15^L    // кредиты за следующий уровень героя (свой у каждого)
```
Bulk ×1 / ×10 / MAX. Уровень — у каждого героя свой.

**Все герои — дамагеры, но с разными профилями статов** (не танк/саппорт). Комбинируешь профили под свой билд:

| Герой | Профиль роста | Открытие |
|---|---|---|
| ⚔️ Воин | +HP, +Урон (крепкий балансный) | стартовый |
| 🎯 Стрелок | +Крит-шанс, +Крит-урон | ростер/Магазин |
| 🔮 Маг | +AoE, +статус-эффекты | ростер/Магазин |
| ... | (ещё профили — контент) | |

- **Комбо/синергия:** каждый герой тянет свой стиль (крит-бёрст / статус-DoT / стабильный урон) — состав команды = билд.
- **Свежий старт:** открыл нового героя → качаешь с нуля = ранний дофамин снова (замена «перерождению»).
- У каждого героя — **свой набор снаряжения** (8 слотов, см. раздел 4).
- Разблок героев: за прогресс глав / осколки / в Магазине за Ядра (как в референсе — коллекция).

## 6. Данжи 🗝️ (вкладка, тайм-гейт)

- **Дроп:** Ядра ⬢ + редкий/повышенной редкости шмот + кредиты.
- Разные данжи (по дню/типу) — повод заходить ежедневно.

## 7.5 Взято из референса (Path of Kings)

- **Таблица дропа per-локация с %** редкостей, лучше с глубиной (см. 4.3).
- **Сундуки-вехи по прогрессу:** вдоль прохождения глав — сундуки за пороги (открыл ✓ / заперт). Дают шмот/Ядра/героев. Стимул «дожать до сундука».
- **Экран-карта кампании:** вертикальный путь локаций с маркером «ТЫ», превью монстров и таблицы дропа каждой локации, прогресс-бар главы (напр. 30/50 килов).
- НЕ берём агрессивный P2W (VIP-пейволы, no-ads-гейты) — референсу за это ставят 3.0.

## 9.5 Графика и вид — 2.5D перспектива (утв. 2026-07-13, как в референсе)

**НЕ 3D. 2.5D в Canvas 2D:** 2D-спрайты (билборды) на псевдо-перспективной «выпуклой» земле + параллакс-фон. Как в Path of Kings (см. image #6).

**Двухфазный бой (утв. 2026-07-13):**
1. **Подход (вид от 1-го лица):** между врагами — FP-«перегон», монстр бежит от горизонта к точке остановки (рука с оружием, дорога несётся, декор-здания уходят В БОК вдоль обочины, тропа-центр свободна).
2. **Файт (камера на команду из 3):** монстр доходит до точки и ОСТАНАВЛИВАЕТСЯ (не доходя до героев) → камера показывает 3 героев, они бьют врага, цифры урона/КРИТ, HP врага → 0. Умер → следующий, снова с подхода.
(Реализовано в мокапе `Восхождение.html` через классы `.mode-approach` / `.mode-fight`.)

Декор-здания: рождаются СБОКУ от дороги, уходят прямо В БОК к краям экрана (не вниз, не на героев), одинаковая скорость у всех, глубина = только размер (ближе крупнее).

- **Камера позади команды**, дорога уходит вдаль. **3 героя на переднем плане** (крупные, лицом к камере), враги идут сверху по дороге и **растут по мере приближения**.
- **Земля** — залитая «выпуклая» фигура (градиент) + дорожка; рисуется кодом.
- **Параллакс-фон** локации: 2–4 слоя (небо/облака → дальние горы → средний план → кусты). Медленный сдвиг для глубины.
- **Глубина = масштаб:** спрайт масштабируется по Y-позиции (ближе=крупнее). Y-сортировка отрисовки.
- **Спрайты ФРОНТ/¾** (видим лицо), НЕ строгий профиль → легче для AI-генерации, не нужно зеркалить фейсинг.
- Всплывающие цифры урона, «КРИТ!», HP-бар над врагом.

**Пайплайн из BD** (`project_sprite_sheet_pipeline`). **Минимальная анимация:** лист 2 кол × 3 ряда (idle/walk/attack, по 2 кадра), ячейка 256 → PNG 512×768. Death — в коде (затухание+падение). Прозрачный фон.

Ассеты: см. `docs/idle_assets.txt`. Шмот слоями на герое — позже.

## Решения (утв.)

- Киберпанк, side-view, AI-спрайты, минимальная анимация.
- Локация = 3 главы; глава = 10 врагов; элита (10-й) + босс (конец блока). Смерть → −1 глава.

- **Навигация = 5 вкладок (v0.4):** Герои · Карта · **Бой**(центр) · Боссы · Магазин. Данжи пока не в навигации (fold-in TBD). Статы за `[!]`/аватар.
- Взято из референса: дроп-таблица % per-локация (лучше с глубиной), сундуки-вехи, экран-карта кампании.
