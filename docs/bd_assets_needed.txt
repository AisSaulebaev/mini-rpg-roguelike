===== PIPELINE ИНТЕГРАЦИИ (когда PNG готовы) =====

1. Кириллические оригиналы → originals\ (gitignored).
2. ASCII-копии → games/bd/img/ (например icon_coin.png, без пробелов
   и кириллицы в имени).
3. После замены/добавления — бамп ?v=N в src этого файла (iOS
   WebView держит старое).
4. Бамп style.css?v=... и game.js?v=... в games/bd/index.html.
5. Для UI-иконок: правлю cardIconHtml() в game.js — эмодзи-fallback
   падает автоматически.
6. Для castle-юнитов: расширяю UNITS, enemyPoolForWave('castle'),
   loadUnitImage() в init, балансовые формулы. Босс w15 — ветка
   startBattle() по state.currentLocation.
7. Для castle-окружения: castleSideDecor/castleFieldDecor по образцу
   cave + ветка в rebuildBackdrop.

Время интеграции после получения арта: ~1ч на UI-иконки, ~2-3ч на
castle-локацию.
