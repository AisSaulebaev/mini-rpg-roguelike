 IDLE RPG «Восхождение» — СПИСОК АССЕТОВ (v0.1, 2026-07-12)
 Киберпанк, 2D side-view. Пайплайн спрайт-листов переиспользуем из BD.
=====================================================================

ФОРМАТ СПРАЙТ-ЛИСТА (минимальная анимация)
---------------------------------------------------------------------
- Сетка 2 колонки x 3 ряда, ячейка 256x256 px -> PNG 512x768 px.
- Ряды (сверху вниз):
    ряд 0 = IDLE   (2 кадра: A/B, лёгкое "дыхание"/парение)
    ряд 1 = WALK   (2 кадра: шаг A/B)   [для стационарных — оставить = idle]
    ряд 2 = ATTACK (2 кадра: замах/выстрел A/B)
- DEATH не рисуем — в коде затухание + падение.
- Фон ПРОЗРАЧНЫЙ (без шахматки). Каждый кадр отцентрирован в ячейке,
  ноги/низ фигуры на ОДНОЙ линии по всем кадрам (иначе прыгает).
- РАКУРС: ФРОНТ / ¾ (лицом к камере), НЕ строгий профиль. Вид игры 2.5D
  (камера позади команды, враги идут сверху по дороге). Герои и враги
  смотрят примерно на камеру. Единый стиль: киберпанк, неон, тёмная база.

ПАЛИТРА/ТОН
---------------------------------------------------------------------
- Тёмный индустриальный фон-тон + неоновые акценты (циан/маджента/жёлтый).
- Враги — БЕЗЛИКИЕ МАШИНЫ (дроны/боты/турели). НЕ люди, НЕ форма, НЕ
  символика реальных структур. (см. правило безопасности в дизайн-доке).

=====================================================================
 ПАРТИЯ 1 — ядро + Локация 1 «Трущобы» (главы 1–3)
=====================================================================

CORE (нужны всегда)
[ ] hero            — герой-протагонист, кибер-курьер/боец. смотрит вправо.
                      idle/walk/attack.
[ ] drone_companion — дрон-компаньон, парит рядом с героем. idle(парение)/
                      attack(выпад/выстрел). walk = idle.

ВРАГИ — Трущобы
[ ] jammer_drone    — мелкий дрон-глушилка, летающий. базовый враг.
                      idle(парение)/walk(подлёт)/attack(разряд).
[ ] shock_bot       — наземный шокер-бот, ближний бой. базовый враг.
                      idle/walk/attack(удар-разряд).
[ ] elite_assault   — ЭЛИТА (10-й враг глав 1 и 2). крупнее, бронированный
                      штурм-дрон. idle/walk/attack.
[ ] boss_overseer   — БОСС ЛОКАЦИИ (10-й враг главы 3): «Дрон-надзиратель».
                      большой, внушительный. idle/walk/attack.
                      (можно позже дать ему свой лист 3x3, пока — 2x3.)

ФОНЫ (2.5D — ПАРАЛЛАКС-СЛОИ, не одна картинка)
[ ] bg_slums_sky    — дальний слой: небо/облака/силуэт города (широкий, тянется).
[ ] bg_slums_far    — средний: неон-высотки/вывески трущоб.
[ ] bg_slums_near   — ближний: мусор, провода, бордюр у дороги.
    (дорога/выпуклая земля рисуется КОДОМ, отдельный ассет не нужен.)
    Каждый слой PNG с прозрачностью, горизонтальный, ~1280 шир. Медленный
    сдвиг слоёв в коде даёт глубину. Можно начать с 1-2 слоёв.
